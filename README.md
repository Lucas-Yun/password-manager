# PasswordManager 密码查询系统

基于 C++ 开发的本地密码管理系统，支持用户注册/登录、密码加密存储、多账号管理、管理员后台等功能。

## 功能特性

### 普通用户

| 功能 | 说明 |
|------|------|
| 添加信息 | 输入平台名称、多个账号（支持多账号）、密码，支持随机强密码生成 |
| 查询信息 | 输入平台名称，显示账号和密码，30秒后自动返回 |
| 删除信息 | 输入平台名称，删除该平台的所有信息 |
| 修改信息 | 修改已有平台的账号或密码，支持随机密码生成 |
| 平台列表 | 查看所有已保存的平台名称及账号数量 |
| 导出数据 | 将密码数据导出为明文备份文件（需妥善保管） |

### 管理员

| 功能 | 说明 |
|------|------|
| 批量添加用户 | 一次性添加多个用户 |
| 删除用户 | 删除指定用户及其所有数据 |
| 查看用户列表 | 查看所有普通用户 |
| 修改用户密码 | 修改指定用户的密码（不可查看） |
| 初始化系统 | 清空所有用户数据和密码信息 |

### 安全特性

- 密码输入隐藏显示（`*`）
- 数据盐值+多字节密钥流加密本地存储（抗频率分析、相同明文不同密文）
- 用户数据完全隔离（A 用户无法访问 B 用户数据）
- 管理员不可查看普通用户的密码隐私
- 注册需管理员密码验证，防止恶意注册
- 密码强度实时检测（弱/中/强）
- 支持生成随机强密码

## 环境要求

- **操作系统**: Windows
- **编译器**: MinGW g++ (支持 C++11)
- **依赖库**: Windows API (`windows.h`, `conio.h`)

## 编译与运行

### 使用 Makefile

```bash
# 编译
make

# 运行
make run

# 清理编译产物
make clean
```

### 手动编译

```bash
g++ -std=c++11 -Wall -finput-charset=UTF-8 -Iinclude \
    src/main.cpp src/crypto.cpp src/file_io.cpp src/user.cpp src/menu.cpp \
    -o password_manager.exe
```

### 运行

```bash
./password_manager.exe
```

## 使用说明

### 首次使用

1. 运行程序，系统自动创建管理员账户
2. 选择「2、注册」，输入管理员密码 `admin_root`
3. 输入用户名和密码完成注册
4. 选择「1、登录」使用注册的账户登录

### 管理员登录

- **用户名**: `Admin`
- **密码**: `admin_root`

### 数据存储

- 用户数据: `data/users.dat`（加密存储）
- 用户密码: `data/<用户名>.dat`（加密存储）
- 数据目录在程序首次运行时自动创建

## 模块说明

| 模块 | 文件 | 职责 |
|------|------|------|
| 加密模块 | `crypto.h/cpp` | 盐值+多字节密钥流加密解密、密码隐藏输入 |
| 文件模块 | `file_io.h/cpp` | 用户数据与密码数据的加载/保存 |
| 用户模块 | `user.h/cpp` | 用户注册、登录验证 |
| 菜单模块 | `menu.h/cpp` | 界面展示、业务逻辑、输入处理 |
| 类型定义 | `types.h` | 全局常量与数据结构 |
